# Proyecto de Análisis y Predicción de Precios de Casas

## Descripción General
Este proyecto tiene como objetivo analizar y predecir los precios de casas utilizando técnicas de minería de datos y modelos de regresión. Se basa en el conjunto de datos "House Prices: Advanced Regression Techniques" de Kaggle, que contiene diversas características de propiedades, incluyendo ubicación, tamaño y calidad constructiva. El análisis incluye exploración de datos, ingeniería de características y la evaluación de modelos predictivos para seleccionar el más eficiente.

## Objetivos del Proyecto
1. **Realizar un Análisis Exploratorio de Datos (EDA)** para comprender la estructura y distribución de los datos.
2. **Identificar las variables más relevantes** para la predicción del precio de las casas.
3. **Agrupar las casas en diferentes categorías** según características clave como calidad y área habitable.
4. **Desarrollar modelos de regresión** para predecir los precios de las casas.
5. **Garantizar la reproducibilidad del análisis**, asegurando que el conjunto de entrenamiento y prueba sea consistente.

## Secciones del Análisis

### **1. Análisis Exploratorio de Datos**
- Exploración de la estructura del dataset.
- Identificación de valores faltantes y estrategias de imputación.
- Análisis de distribución de variables numéricas y categóricas.
- Pruebas de normalidad y correlación entre variables.

### **2. Análisis de Modelos Generados**
- Implementación de diferentes modelos de regresión.
- Evaluación de los residuos y análisis de errores.
- Explicación de los razonamientos detrás de cada modelo utilizado.

### **3. Aplicación de Modelos al Conjunto de Prueba**
- División del dataset en entrenamiento y prueba asegurando reproducibilidad.
- Aplicación de los modelos entrenados en los datos de prueba.

### **4. Explicación de Resultados**
- Análisis del desempeño de cada modelo utilizando métricas como RMSE y R².
- Interpretación de los resultados y relación con las variables seleccionadas.

### **5. Comparación de Modelos y Selección del Mejor**
- Comparación del desempeño de los modelos utilizados.
- Selección del mejor modelo basado en precisión y generalización.
- Justificación de la elección del modelo final para la predicción del precio de las casas.

## Tecnologías Utilizadas
- **Python** (pandas, numpy, matplotlib, seaborn, scikit-learn)
- **Machine Learning** (Regresión Lineal, K-Means Clustering, Ingeniería de Características)
- **Visualización de Datos** (Gráficos de correlación, histogramas, scatter plots)
